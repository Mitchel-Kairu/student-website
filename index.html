HTML as One of My Core Skills

What is HTML?

HTML (HyperText Markup Language) is a markup language used to create the structure of web pages. It defines the content and meaning of elements on a webpage.

Structural Elements

<html> defines the root of the document.

<head> contains metadata.

<body> contains visible content.

<section> groups related content.

<article> represents independent content.

<header> defines introductory content.

<footer> defines closing content.

Text Formatting Elements

<h1> to <h6> define headings.

<p> defines a paragraph.

<strong> indicates important text.

<em> indicates emphasized text.

Lists

<ul> unordered list.

<ol> ordered list.

<li> list item.

Links

<a> is used to create hyperlinks using the href attribute.

Images

<img> is used to display images. It requires src and alt attributes.

Tables

<table> defines a table.

<tr> table row.

<th> table header.

<td> table data.

Forms

<form> collects user input.

<input> input field.

<label> labels a form control.

<textarea> multi-line input.

<button> clickable button.
